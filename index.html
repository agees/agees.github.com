1st : Start Up!

Feb 9th, 2012

Railsやってみよう

なにはともあれRailsやってみよう

今日やること

  • vimに触ってみる
  • Rails動かしてみる
  • gitを使ってみる

環境設定

  • Install vim
    vim
    vimをダウンロードしてインストールする。

  • コマンドライン環境を作る
    windowsにlinuxライクな使いやすいコマンドライン環境を作る。

    • Console2 + nyaos
      まずはConsol+nyaosをイン
      Console2
      nyaos
      Console2を開い
      Edit->Settings->Tab->Shell
      にnyaos.exeのパスを指定すればOK。
    • nerdtree
      その後にnerdtreeをインストする。
      nerdtreeとはコンソールから使えるエクスプローラのようなvim拡張。便利。
      nerdtree
      落としたzipを解凍/vim/vimfilesに配置する。

  • githubアカウントの作成 ソース管理にはgithubを使う。Railsのインストール時にアカウントが必要に
    なるので以下でアカウントを作っておく。
    github

  • Install Rails
    Railsをインストールする。
    RailsInstaller
    インストール後にgithubアカウントを登録する。

Railsを動かしてみる

とりあえずRailsを動かしてみる。まずはバージョンの確認。

$ rails -v
Rails 3.1.1

新規アプリケーションを作る。

$ rails new HelloRails

動作確認してみる。

$ rails s
